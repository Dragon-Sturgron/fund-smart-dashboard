# 基智盘 v2.5.1 盘中持仓自动更新版

- 持有份额保持固定，两位小数。
- 持有金额 = 持有份额 × 盘中估算净值。
- 持有收益 = 持有金额 − 投入本金。
- 设置页打开期间每 60 秒自动刷新，重新聚焦页面也会刷新。
- 手动修改持有金额/收益后，点击“保存全部持仓”重设持仓基准。
- KV 仍只保存基金代码。

# 基智盘 v2.5.1 保存重算修正版

本版修复“保存全部持仓”点击后持有份额不更新的问题。

- 修改持有金额或持有收益时仅保留输入，不提前刷新页面。
- 点击“保存全部持仓”后，并行重新读取每只基金的上一确认净值。
- 持有份额 = 持有金额 ÷ 确认净值。
- 持有份额四舍五入保留两位小数。
- 持有收益用于计算投入本金和平均持仓成本，不直接改变持有份额。
- KV 仍然只保存基金代码。

部署配置保持不变：输出目录 `public`，构建命令 `npm run check`。
