# 基智盘 v2.7.0 KV 持仓同步与每日零点更新版

- 基金代码、持有金额、持有收益、实际持有份额同步到 EdgeOne KV。
- 不同设备打开设置页时可从 KV 恢复基金与个人持仓。
- 实际持有份额可直接修改，四舍五入保留两位小数。
- 自动更新固定为每天北京时间 00:00 一次；当时未打开网页，会在当天首次打开设置页时补更新。
- 每日自动更新只改变持有金额和持有收益，不修改实际持有份额。
- 点击“保存全部持仓”会立即保存并同步到 KV。

构建配置：Other / 根目录 ./ / 输出目录 public / 构建命令 npm run check / KV变量 FUND_KV。
