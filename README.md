# 基智盘 v2.5.1 份额两位小数版

本版基于 v2.5.1 份额修正版，将持有份额统一改为四舍五入保留两位小数。

## 调整

- 持有份额计算：`持有金额 ÷ 上一确认净值`。
- 计算结果使用四舍五入，统一保留两位小数。
- 已存在的多位小数份额会在首次进入设置页时自动转换为两位小数。
- 页面、成功提示和本机存储中的份额均统一为两位小数。
- 保留基金名称自动识别、“添加并继续”和旧份额自动修正。
- KV 仍只保存基金代码。
